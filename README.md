# VerificationTask
Для ввода данных созданна windowsForm с элементами TextBox, а так же кнопками расчета, сброса и сохранения.

На всех полях стоит проверка на ввод только целочисленных данных и возможность стирать введенные символы.
После нажатия кнопки "Расчитать" пользователь видит таблицу с расчетами электроэнергии, холодной и горячей воды, а так же тарифы, нормативы и показания.

Нажимая кнопку "сохранить" данные сохраняются в базу данных

Нажимая кнопку "сбросить" очищается таблица и поля для ввода
