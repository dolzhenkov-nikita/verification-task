# VerificationTask
Для ввода данных создана windowsForm с элементами TextBox, Label, Button и dataGridView.

textBox используются для ввода показаний и количества жильцов

label для пояснений к полям textbox

button элементы используются для запуска функций расчета, сброса или сохранения

dataGridView таблица служит для вывода результатов расчетов

В проекте присутсвует база данных, которая должна работать по пути относительно .exe файла после сборки.
Если база незаработает, в файле from1 закоментирован код создания требуемых таблицы
